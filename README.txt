This is Kivi Dollin's Shopping Cart Program. 
The program shows you items for sale. Each item has a price and the ability to see a Description and Reviews for the item. The cart keeps track of how many items you have, and if you want more than one of each item. 
To see your total, click CHECKOUT, or UPDATE TOTALS. 


Languages Used: Javascript, CSS, HTML
Framework: Angular.js

To Run: Simply open ShoppingCartFinal.html in a browser
